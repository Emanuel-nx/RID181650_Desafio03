# Gerenciador de Tarefas

Board interativo para gerenciar tarefas com prioridades, categorias e controle de conclusão.

## Funcionalidades

- Carrega tarefas iniciais automaticamente
- Adiciona novas tarefas (nome, categoria e prioridade)
- Marca tarefa como concluída (nome riscado + ícone ✔)
- Exclui tarefas individualmente
- Contadores atualizados em tempo real (total, pendentes, concluídas)

## Como usar

Abra o arquivo `index.html` diretamente no navegador — sem dependências, sem instalação.

## Tecnologias

- HTML5, CSS3, JavaScript (vanilla)
- Fonte: [Rubik](https://fonts.google.com/specimen/Rubik) via Google Fonts
